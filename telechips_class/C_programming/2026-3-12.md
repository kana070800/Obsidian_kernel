	`int b(int a[4])`처럼 선언시 b 내부에서 a는 포인터로 인식 >> sizeof(4) == 4
	`int c(int f(int)) >> int c(int (*)(int))
	`int b(int x)[4] >> int *b(int x) or int(*b(int x))[4]
	`int c(int x)(int) >> int (*c(int x))(int)`
	`int c[4](int) >> int (*c[4])(int)`

typedef > 변수이름 위치에 재정의할 타입이름
	`typedef int INA[5] >> 배열타입 INA`
	`typedef int(*INAP)[5] >> 배열 포인터타입 INAP`
	`#define을 통해 타입 정의하면 * 사용시 주의 필요(권장하지 않음)

`#define 을 통해 매크로 함수 정의시 연산 우선순위 오류 발생 가능성 고려 >> 괄호 적극 활용`
	`#define 은 \으로 여러줄 확장 가능`

`#line b>> 해당 위치를 b번 라인으로 지정`
